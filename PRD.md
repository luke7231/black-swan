# **1\. Executive Summary**

발레매니아(2002년 창업)는 국내 유일의 발레 전문 구인구직 플랫폼으로, 20년 이상 독점 지위를 유지하고 있다. 그러나 모바일 대응 부재, 알림 기능 없음, 구식 UX로 인해 사용자 불편이 크다.

본 서비스는 '발레 강사를 위한 실시간 알림 앱'으로 시작해 점진적으로 자체 구인구직 플랫폼으로 성장한다. 초기에는 발레매니아 공고를 링크로 연결하여 콘텐츠 구축 없이 즉시 가치를 제공하고, 강사 베이스가 확보되면 기업 직접 유치로 전환한다.

| 구분 | 발레매니아 (경쟁사) | 우리 서비스 |
| :---- | :---- | :---- |
| 창업 | 2002년 | 신규 |
| 플랫폼 | 웹 전용, 모바일 미지원 | 웹 \+ iOS/Android 앱 |
| 알림 | 없음 (직접 방문해야 확인) | 지역·조건별 푸시 알림 |
| 대타 매칭 | 게시판 수준 | 실시간 매칭 알림 |
| 강사 요금 | 39,000원/월 | 영구 무료 |
| 기업 요금 | 복잡한 add-on 구조 | Simple 3 티어 플랜 |
| 추정 월매출 | 2,500만\~3,000만원 | Year 2 목표: 5,000만원+ |

# **2\. Problem Statement**

## **2.1 강사 (구직자) 문제**

* 원하는 조건(지역·요일·시간대)의 공고가 올라와도 알림이 없어 직접 사이트를 방문해야 한다.

* 대타 요청은 당일 급구가 많으나 실시간 알림이 없어 학원과 강사 모두 놓치는 경우가 빈번하다.

* 채용정보 열람에 월 39,000원을 내야 하는 진입장벽이 있다.

* 모바일에서 사이트 사용이 불편해 이동 중 확인이 어렵다.

## **2.2 기업 (구인자) 문제**

* 프리미엄 배너, 급구, 우대, 자동점프, 강조효과 등 서비스가 파편화되어 있어 결제가 복잡하다.

* 강사 이력서 열람 구독(기업정회원)과 채용공고 노출 서비스가 분리되어 있어 혼란스럽다.

* 지원자 관리, 메시지, 계약서 기능이 없어 채용 이후 프로세스는 카카오톡 등 외부 도구에 의존한다.

# **5\. 제품 전략**

3단계 성장 전략으로 치킨앤에그 문제를 해결하고 발레매니아를 점진적으로 대체한다.

| Phase 1 | 알림 레이어 — 강사 선점  |  0\~6개월 발레매니아 공고를 링크로 연결. 알림 기능으로 즉각적인 가치 제공. 강사 베이스 구축. 지역·요일·시간대 조건 설정 → 조건 맞는 공고 푸시 알림 대타 실시간 매칭 알림 (당일 급구 전용) 발레매니아 공고는 딥링크로 연결 (스크래핑 아님, 법적 안전) 카카오 로그인, 무료 가입 — 강사 진입장벽 제로화 |
| :---: | :---- |

| Phase 2 | 자체 플랫폼 — 기업 직접 유치  |  6\~18개월 강사 수천 명 확보 후 기업 콜드 아웃리치 시작. 자체 공고 비중 확대. 기업 무료 공고 등록 (초기 3개월) 지원자 관리 대시보드 발레매니아 링크 비중을 자체 공고로 점진 대체 49,000원 유료 플랜 출시, 프리미엄 배너 광고 상품 출시 |
| :---: | :---- |

| Phase 3 | B2B SaaS — 학원 운영 도구  |  18\~24개월 구인구직을 넘어 학원 운영 플랫폼으로 확장. 이탈 방지, LTV 극대화. 수강생 관리, 출결, 정산 기능 강사 계약서 템플릿 및 전자서명 강사 포트폴리오 페이지 (영상, 수료증) 앱 내 1:1 채팅 기능 출시 강사 유료 프리미엄 기능 (우대 인재 노출 등) 검토 |
| :---: | :---- |

# **6\. 기능 명세**

## **6.1 Phase 1 — 핵심 기능 (MVP)**

### **6.1.1 강사 회원 (무료)**

* 카카오 / 애플 / 이메일 회원가입 및 로그인

* 관심지역 2개 무료 제공 (시/구 단위) — 친구 초대 1명당 \+1개 추가 (상한 없음)

* 알림 조건 설정: 관심지역, 요일, 수업 시간대 (오전/오후/저녁), 수업 유형 (정규/대타)

* 알림 수신: 조건 매칭 시 푸시 알림 \+ 앱 내 알림

* 공고 상세보기: 원본 사이트 링크로 이동 (발레매니아 / 이상댄스 / 호호요가 / 댄스잡)

* 대타 알림 구독: 당일 대타 요청 즉시 수신

* 관심 공고 저장 (북마크)

* 친구 초대 링크 생성 및 초대 현황 확인

### **6.1.2 멀티소스 공고 수집 \+ AI 분류 파이프라인**

4개 사이트에서 발레 공고를 수집하여 통합 DB로 구성한다. 발레 외 종목(바레, 현대무용, 한국무용, 방송댄스 등)은 수집 단계에서 전량 제거한다.

### **수집 대상 사이트**

| 사이트 | 로그인 필요 | 수집 방식 | 특이사항 |
| :---- | :---- | :---- | :---- |
| 발레매니아 (balletmania.com) | 부분 (상세) | Cheerio \+ EUC-KR 변환 | 딥링크 연결 |
| 이상댄스 (esangdance.net) | 불필요 | Cheerio | 카테고리 필터 활용 |
| 호호요가 (hohoyoga.com) | 불필요 | Cheerio | 지역 구조화 데이터 풍부 |
| 댄스잡 (dancejob.net) | 부분 (상세) | Cheerio | 분야/지역 구조화 |

### **AI 분류 파이프라인**

수집된 원문(제목 \+ 본문)을 LLM에 1회 호출하여 구조화 데이터로 변환. 키워드 룰로 처리 가능한 항목은 LLM 없이 처리하고, 애매한 케이스만 LLM fallback.

| 분류 항목 | 방법 | 출력값 |
| :---- | :---- | :---- |
| 발레 여부 필터 | 키워드 룰 (1순위) | 통과 / 드랍 — 바레·무용·힙합 등 전량 제거 |
| 수업 대상 | 키워드 룰 → LLM fallback | 유아 | 성인 | 입시 | 혼합 |
| 공고 유형 | 키워드 룰 (대강/대타/급구) | 정규 | 대타 |
| 지역 정규화 | 파싱 \+ 주소 표준화 | 시/도 \+ 시/군/구 표준 형식 |
| 요일 | 정규식 파싱 | \[월, 화, 수, …\] |
| 시간대 | 키워드 룰 | 오전 | 오후 | 저녁 | 협의 |
| 급여 | 정규식 파싱 | 숫자 (만원) or null |

### **중복 학원 판별 (크로스사이트 dedup)**

* 같은 학원이 여러 사이트에 공고를 올린 경우 하나의 entity로 묶어 표시

* 판별 기준: 학원명 fuzzy matching \+ 지역 일치 → 1차 후보 선정

* 전화번호 / 연락처 동일 시 확정 처리

* 동일 학원으로 판별된 공고는 '여러 사이트에 올라온 공고' 뱃지 표시

### **6.1.3 대타 게시판**

* 대타 요청 등록: 날짜, 시간, 지역, 수업 내용, 페이 입력

* 등록 즉시 해당 지역 강사에게 푸시 알림 발송

* 24시간 자동 만료

* 연락 방식: 공고에 기재된 연락처 직접 연락 (채팅 기능은 Phase 3 이후)

## **6.2 Phase 2 — 자체 플랫폼**

### **6.2.1 기업 회원 — 무료**

* 기업 인증 (사업자번호 확인)

* 채용공고 직접 등록 (무제한)

* 지원한 강사 지원자 확인 및 연락처 열람

### **6.2.2 기업 회원 — 유료 플랜 (49,000원/월)**

* 우선 채용 노출: 채용공고 리스트 최상단 고정

* 급구 채용 무제한: 건수 제한 없이 급구 공고 등록

* 강사 이력서 직접 열람 및 채용 제의 (무제한)

* 대타 알림 서비스: 대타 요청 시 해당 지역 강사 전원 즉시 푸시

* 즐겨찾기 (인재풀): 관심 강사 저장 및 관리

* 매칭 알림: 설정 조건 맞는 강사 신규 등록 시 기업에 알림

### **6.2.3 프리미엄 배너 광고 (별도 상품, 99,000원/월)**

* 메인 페이지 최상단 프리미엄 배너 슬롯 노출

* 채용공고 리스트 최상단 배너 노출

* 구독 플랜과 무관하게 별도 구매 가능

* 발레매니아 동급 상품 대비 약 20% 저렴 (발레매니아: 120,000원/월)

### **6.2.4 강사 이력서**

* 이력서 등록: 사진, 전공, 경력, 취득 자격증, 가능 지역/요일

* 이력서 공개/비공개 설정

* 기업 제의 수신함

## **6.3 Phase 3 — 학원 운영 도구**

* 수강생 관리: 등록, 출결, 결제 이력

* 강사 계약서: 표준 템플릿 제공 및 전자서명

* 정산 자동화: 수업 건수 기반 강사 페이 계산

* 강사 포트폴리오 페이지 (퍼블릭 URL 공유 가능)
